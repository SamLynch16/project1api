API Documentation

1. Get All Pokémon

URL:/api/getAllPokemon

Methods:GET/HEAD

Description: Retrieves a list of all Pokémon in the Pokédex.

Request Example: GET /api/getAllPokemon

Response Example:

{
  "success": true,
  "data": {
    "pokedex": [ /* Array of all Pokémon data */ ]
  }
}

2. Get Pokémon by Name

URL: /api/getPokemonByName

Methods: GET/HEAD

Description:Retrieves a specific Pokémon by its name.

Query Parameters:

  • name (required): The name of the Pokémon (e.g., Pikachu)

Request Example: GET /api/getPokemon?name=Pikachu

Response Example (Success):

{
  "success": true,
  "data": {
    "id": 25,
    "name": "Pikachu",
    "type": ["Electric"]
  }
}

Response Example (Error - Pokémon Not Found):

{
  "success": false,
  "error": "Pokémon with name 'Pikachu' not found."
}

3. Get Pokémon by ID

URL:/api/getPokemonById/:id

Methods:GET/HEAD

Description: Retrieves a specific Pokémon by its unique ID.

Route Parameters:

  • id (required): The ID of the Pokémon (e.g., 25)

Request Example: GET /api/getPokemonById/25

Response Example (Success):

{
  "success": true,
  "data": {
    "id": 25,
    "name": "Pikachu",
    "type": ["Electric"]
  }
}

Response Example (Error - Pokémon Not Found):

{
  "success": false,
  "error": "Pokémon with id '25' not found."
}

4. Get Pokémon by Type

URL: /api/getPokemonByType

Methods: GET/HEAD

Description: Retrieves Pokémon filtered by type.

Route Parameters:

  • type (required): The type of Pokémon (e.g., Fire)

Request Example:GET /api/getPokemonByType/Fire

Response Example (Success):

{
  "success": true,
  "data": {
    "pokedex": [ /* Array of filtered Pokémon data by type */ ]
  }
}

5. Add Pokémon

URL: /api/addPokemon

Method: POST

Description: Adds a new Pokémon to the database.

Request Body (JSON):

{
"id": 150,
"name": "Mewtwo",
"type": ["Psychic"]
}

Response Example (Success):

{
"success": true,
"message": "Pokémon added successfully."
}

6. Add Move

URL: /api/addMoves

Method: POST

Description: Adds a new move to a Pokémon.

Request Body (JSON):

{
"pokemonId": 25,
"move": "Thunderbolt"
}

Response Example (Success):

{
"success": true,
"message": "Move added successfully."
}
